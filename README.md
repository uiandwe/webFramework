# webFramework
webFramework


### run
node app.js

### test
npm test

### doc
##### $ grunt jsdoc:cliet
##### $ grunt jsdoc:server

### 폴더 설명

- /app      : js / css / SPA 사용시 view파일
- /config   : 서버 셋팅 관련
- /meta     : 설정값 및 각종 코드 / 언어셋
- /middles  : 미들웨어
- /model    : 디비모델
- /routes   : 디비모델
- /static   : static 파일 및 업로드 파일 저장소
- /test     : mocha tester 파일
- /utils    : 모델/route에 쓰이는 각종 유틸 모음
- /views    : ejs
